# quickjs for Java

**QuickJS Java** is a repackaging of the [quickjs-wasm-java](https://github.com/StefanRichterHuber/quickjs-wasm-java) package
by Stefan Richter-Huber, so I can take it in a slightly different direction.

Building can be done with maven, but as Maven is both slow and opuqae, there are several shell scripts
to build the various stages of the project entirely independently of Maven. Start with `all.sh`
